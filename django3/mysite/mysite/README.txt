This directory contains python files for running the site.

Here is what is contained:

Directories:
	static: contains static image files as well as static giles needed for using bootstrap template.
	_pycache_: again, caches generated from running the server. Kept just in case. 
	_init_ : generated automatically when Django project was created
Files:
	settings.py: python file where settings are adjusted for site. 
	urls.py: python file containing website link route information.
	wsgi.py: generated automatically when Django project was created. This has not been altered.
