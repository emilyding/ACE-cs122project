These are static images used in the website.

	best_cuisines: generated by matplotlib, this graph shows the top ten cuisines across the 55 cities.
	worst_cuisines: generated by matplotlib, this graph shows the top ten worst cuisines across the 55 cities.

	star_reviews_all_cities: generated by matplotlib, this graph shows average #of revews/restaurant for each star rating
	star_restaurants_all_cities: generated by matplotlib, this graph shows number of restaurant per star rating
